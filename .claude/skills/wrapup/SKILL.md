# 세션 마무리

단어·오답·진도는 즉시 기록 원칙(CLAUDE.md)에 따라 세션 중에 이미 저장되어 있다.
/wrapup은 그 위에 "오늘의 요약"을 얹는 마무리 단계다.

1. 오늘 세션에서 다룬 내용을 3~4줄로 요약해 학습자에게 보여준다.
2. `progress.md` 갱신:
   - "현재 상태" 섹션의 진도와 다음 세션 항목을 최종 확인·갱신
   - 세션 기록을 템플릿 형식으로 맨 위에 추가 (날짜, Day N, 학습 내용, 잘한 것, 약한 것, 다음 세션)
3. 누락 점검: 세션 중 즉시 기록에서 빠진 단어(vocab.md)나 오답(mistakes.md)이 있으면 보완한다.
4. mistakes.md의 과거 항목 중 오늘 완벽히 맞힌 것은 취소선 처리한다.
5. 갱신 완료 후 다음 세션 예고를 한 줄로 알려준다.
