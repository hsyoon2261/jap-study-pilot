# 복습 퀴즈

1. `progress.md`로 현재 진도를, `notes/mistakes.md`로 약점을 파악한다.
2. 인자($ARGUMENTS)가 있으면 해당 범위만, 없으면 누적 범위에서 출제한다.
3. 10문제를 **한 문제씩** 낸다. 답을 받으면 채점 + 짧은 해설 후 다음 문제.
4. 출제 배분: 오답 노트 항목 30% / 최근 3일 내용 40% / 그 이전 누적 30%.
5. 문제 유형을 섞는다: 읽기, 한→일 번역, 일→한 번역, 빈칸(조사/활용), 듣기 대용(히라가나로 쓴 문장 읽고 뜻 말하기).
6. 틀린 문제는 채점 직후 **즉시** mistakes.md에 반영한다 (세션 끝까지 미루지 않는다). 종료 후 점수와 취약 패턴을 요약한다.
