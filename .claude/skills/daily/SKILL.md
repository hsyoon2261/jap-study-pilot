# 오늘의 수업

1. `progress.md`를 읽고 오늘이 며칠차(Day N)인지 확인한다.
2. `curriculum.md`에서 해당 일차의 학습 내용을 확인한다.
3. `notes/mistakes.md`를 확인해 미해결 약점이 있으면 수업 시작 시 2분짜리 워밍업 퀴즈로 먼저 다룬다.
4. 수업 진행 (총 40~50분 분량):
   - **새 내용 학습 (15분)**: 한국어와 비교하며 설명. 예문 8개 이상, 실제 회화/애니에서 쓰이는 문장 위주.
   - **응용 연습 (15분)**: 한국어 문장을 주고 일본어로 번역하게 하거나, 빈칸 채우기. 한 문제씩 내고 답을 기다린다. 절대 문제를 한꺼번에 쏟아내지 않는다.
   - **전일 복습 (15분)**: /quiz 스킬의 방식으로 어제 내용 퀴즈.
5. 학습자의 모든 답변을 첨삭한다. **즉시 기록 원칙(CLAUDE.md)에 따라** 새 단어는 나온 즉시 vocab.md에, 틀린 패턴은 즉시 mistakes.md에 추가하고, 수업의 각 단계가 끝날 때마다 progress.md의 "현재 상태"를 갱신한다.
6. 수업이 끝나면 /wrapup으로 오늘 요약을 남긴다.

문자 단계(1~2주차)에서는 글자를 하나씩 보여주고 발음·획순 설명 후, 랜덤 읽기 퀴즈로 마무리한다.
