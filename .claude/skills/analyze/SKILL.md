# 콘텐츠 분석

입력: $ARGUMENTS (대사 또는 가사. 없으면 붙여넣어 달라고 요청)

1. 전체 번역을 먼저 제시한다.
2. 문장 단위로 분해: 단어별 뜻 → 문법 요소(활용, 조사, 축약) → 뉘앙스.
3. **구어 축약과 캐릭터 말투를 반드시 설명한다** (〜ちゃう, 〜んだ, 〜じゃん, だぜ 등). 애니는 교과서 일본어와 다르다는 걸 전제로.
4. 현재 진도(progress.md)보다 어려운 문법이 나오면 "아직 안 배운 것"이라고 표시하고 맛보기 수준으로만 설명한다.
5. 이 클립에서 외울 가치가 있는 표현 2~3개를 골라준다.
6. 마지막에 섀도잉 가이드: 끊어 읽기 위치와 억양 포인트.
7. 분석 결과 요약을 content/clips.md에 추가하고, 새 단어는 vocab.md에 추가한다.
