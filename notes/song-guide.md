# 🎵 노래 작업 표준 (Song Work Guide)

노래 한 곡을 학습 콘텐츠로 만드는 **표준 절차 + 품질 기준**. 100곡+ 일관성을 위해 튜터(클로드)는 매 곡 이 표준을 따른다. (확정 2026-07-20, 학습자 지시)

## 0. 대원칙
- **학습자**: 37세 한국인 개발자, 일본어 초급. 목표 = 자막 없이 애니→드라마→J-pop 이해. **한국어 화자 이점(한자어·어순·조사)을 적극 활용**해 설명.
- **저작권**: 가사 원문 반입은 **학습자 클릭**(앱 우타텐 버튼/붙여넣기)만. 튜터는 가사를 직접 긁거나 채팅에 재현하지 않는다. 데이터는 로그인 뒤(인증 엔드포인트)로만 서빙 — public static 금지.
- **백그라운드 작업**: 학습자가 "보여줘/띄워줘" 하기 전엔 브라우저 안 띄운다. 검증은 코드/API(python·wrangler·curl)로.

## 1. 파이프라인 (순서)
1. **가사 확보** — 학습자가 앱에서 우타텐 원클릭/붙여넣기 (튜터 아님)
2. **자막 취득** — 튜터: `yt-dlp --write-auto-subs --sub-langs ja-orig --sub-format json3 -o data/video-text/%(id)s.%(ext)s <URL>`
3. **소절 재분할** — 튜터: 자막 호흡 + 완전한 문장 단위 (§2)
4. **후리가나(h)** — 전 토큰 로마자 ruby, 자동생성 (§3)
5. **상세(d)·뜻(t)·노트(n)** — 밀도 기준대로 작성 (§4)
6. **싱크(lineTimes)** — forced-align → D1 (§6)
7. **배포** — songs.json은 `git push`(웹 CF 빌드) + D1 times는 `wrangler d1 execute`(즉시). 백그라운드.

## 2. 소절 나누기 규칙 ⚠️
- ❌ **우타텐 줄바꿈 그대로 쓰기 절대 금지** (학습자가 이걸로 크게 고생함)
- 유튜브 ja-orig 자막의 **실제 호흡·타이밍**을 참고해 끊는다
- **완전히 이어지는 문장/절 단위**로 묶기. 중간 토막은 합치고, 다음 줄로 이어지면 `cont:true` (앱에서 → 표시)
- 구문·문법 학습에 유리하게(문장 단위 우선), 단 한 소절이 너무 길면 분할

## 3. 후리가나(h) 규칙 ⚠️
- **모든 토큰(한자 + 가나)에 로마자 rt.** 한자에만 달고 가나를 맨몸으로 두면 안 됨 (초급 학습자용)
- 자동생성: 가사 `漢字（かな）` 표기 → kana→romaji(촉음っ·요음ゃゅょ·장음ー 처리) → `<ruby>surface<rt>romaji</rt></ruby>` (토큰 사이 `<wbr>`, 전각공백 `　`)
- 조사 단독 は/へ/を → wa/e/o 보정
- 스크립트: `scratchpad/fix_marigold_ruby.py` (kana_units + build_h)

## 4. 소절 상세(d) 밀도 기준 ★★ 핵심
반복 아닌 각 소절은 **최소 이 섹션들** (평균 500자+):
- **의역** — 자연스러운 한국어 뜻 + 왜/뉘앙스 (직역 아님)
- **구조** — 어순 대조: `[주어]/[목적어]/[술어]`로 분해해 한국어 병렬 제시
- **단어표** — 단어 / 읽기(히라가나·한글) / 뜻. 한자어는 한국음 사촌 표시
- **문법 ①②③** — 각 문법점 설명 + **예문 1~2개**. 반드시 다룰 것: 조사, 활용(가능형·의지형·て형·과거관형 등), **자·타동사 짝**(離す/離れる, 残す/残る…), 관형절, 문형(〜すぎる·〜ほど·〜ように 등)
- **뉘앙스·표현·문화** — 애니/드라마/회화 어디서 듣는지, 표기 선택의 감정, 문화 배경, 다른 곡과의 재회 어휘
- 발음 팁(촉음·장음·탁음) 필요시
- **반복 사비**: "N번 반복" 참조 + 핵심 차이만 (경량)

## 5. 뜻(t)·노트(n)
- **t**: 자연스러운 의역 (인라인 표시). 어색한 직역 금지
- **n**: 인라인 핵심 한 줄 (단어·문법 키워드 요약)

## 6. 싱크(lineTimes) — 튜터가 자동, 학습자 탭 X
- yt-dlp ja-orig json3 → **단어별 tOffsetMs** → **forced alignment**(Needleman-Wunsch, 글자단위) → lineTimes
- `song_times` D1 upsert (`wrangler d1 execute --remote`). D1 공유라 **배포 없이 즉시** 로컬·라이브 반영
- 스크립트: `scratchpad/forced_align.py`
- 앱 내 **탭 싱크**(admin)는 미세조정 폴백
- (옵션) ffmpeg 설치되면 음원에 직접 forced-align → 프레임 정밀도

## 7. 세트/진도
- 곡 상태 단계: 영상만 → 가사만(붙여넣음) → 해설 완료(lineNotes) → 싱크 완료(lineTimes)
- songs.json은 서버 번들(`app/src/lib/data/songs.json`), 인증된 `/api/songs`로만 서빙

관련 규칙 파일: 이 세션 memory의 `lyric-segmentation-rule`, `retire-legacy-8765`, `deploy-direction`.
